### Compilation

C
```c
gcc -Wall -o outFileName fileName.cpp
```

C++
```c++
g++ -Wall -o outFileName fileName.cpp
```

Haskell
```haskell
ghc -o outFileName fileName.hs
```

Haskell (loading into gchi)
###### After loading into the Haskell interpreter by running "ghci"
```haskell
:l fileName.hs
```
